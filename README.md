# Sky Signals Website 🌤️

A simple weather forecast web application that allows users to check the weather for today and the next two days. Users can enter a city name and see the forecast using **[WeatherAPI](https://www.weatherapi.com/)**.

---

## Features

- Enter a city name to get the weather
- 3-day forecast (today + 2 days ahead)
- Displays:
  - Temperature
  - Weather condition
  - Humidity
  - Wind speed
  - Chance of rain
- Lightweight and user-friendly interface

## How to Use

1. Open `index.html` in a browser  
2. Enter a city name in the input field  
3. Click **Search**  
4. View the weather details for today and the next 2 days

---

## Setup Instructions

1. Sign up at [WeatherAPI](https://www.weatherapi.com/) and get your API key  
2. Replace `YOUR_API_KEY` in your JavaScript file with your key  
3. Open `index.html` in a browser or deploy the project via GitHub Pages
